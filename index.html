El grid se actualiza automáticamente desde tu base de datos de Notion via Make cada 15 minutos.

Cómo funciona la sincronización

1. Completás el campo "Orden en grid" en tu base de datos de Notion

2. Make detecta el cambio y actualiza el archivo grid.json en GitHub

3. Este widget lee el grid.json cada vez que se abre o al hacer "Actualizar"

4. El grid se reordena automáticamente según el número de orden

Links de imagen compatibles

Usá links que terminen en .jpg .png .webp
